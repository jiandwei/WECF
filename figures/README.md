# Figures Directory

This directory is for storing figures and plots.

## Recommended File Formats

- **PDF**: Best for vector graphics (plots, diagrams)
- **PNG**: Good for screenshots and raster images
- **EPS**: Alternative vector format

## Naming Convention

Use descriptive names:
- `figure1_main_results.pdf`
- `figure2_robustness.pdf`
- `plot_descriptive_statistics.pdf`

## Example Usage in LaTeX

```latex
\begin{figure}[H]
    \centering
    \includegraphics[width=0.8\textwidth]{figures/your_figure.pdf}
    \caption{Your Caption}
    \label{fig:your_label}
\end{figure}
```
